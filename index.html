Popup UX Test — имитация KYC Flow

Проверяем как браузер реагирует на разные способы открытия popup

✅ Тест 1: Синхронный popup

window.open() сразу по клику. Должен открыться везде.

⚠️ Тест 2: Async 500ms (быстрый API)

await 500ms → window.open(). Имитация быстрого ответа API.

⚠️ Тест 3: Async 2 сек (нормальный API)

await 2000ms → window.open(). Типичная задержка BB → ChainBridge.

🔴 Тест 4: Async 5 сек (медленный API)

await 5000ms → window.open(). Медленная сеть или холодный старт.

🔴 Тест 5: Async 8 сек (таймаут)

await 8000ms → window.open(). Сервер тормозит.

🔧 Тест 6: Workaround — open сначала, URL потом

window.open('about:blank') сразу → await 3000ms → popup.location = url. Рекомендуемый обход блокировки.

🔄 Тест 7: Redirect (не popup)

await 3000ms → window.location.href = url. Никогда не блокируется. Страница уходит полностью (нажмите "назад" чтобы вернуться).
